# Clamav-rest Signature Monitor

This API is meant to be a sidecar container for [clamav-rest](https://github.com/ajilach/clamav-rest) since the project decided not to implement this feature in the core api.
Feel free to fork or use this alongside `clamav-rest` according to the [LICENSE](./LICENSE.MD)

## Endpoint

* `GET /health/sig`
